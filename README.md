# Sistem Automatizat de Audit Financiar - SQL & Python

### Descrierea Proiectului
Am construit o soluție de audit care combină SQL pentru gestionarea/automatizarea datelor și Python pentru automatizarea raportării. Proiectul identifică automat erorile de sistem, asigură integritatea financiară și generează rapoarte livrabile.

### Scenarii de Audit Verificate
* **Audit Tranzacțional (01):** Detectarea valorilor 'NULL' sau 'ERROR' și identificarea fluxurilor financiare suspecte.
* **Integritatea Fiscală:** Calcularea automată a TVA-ului și prețului total pentru a preveni erorile de operare umană.

### Tehnologii
* **SQL (MySQL)**: Stocarea datelor, Funcții Stocate și Triggers pentru automatizare.
* **Python**: Automatizarea procesului de audit și conectarea la baza de date.
  
### Rezultate
* **Automatizarea**: Eliminarea rulării manuale a interogărilor și a calculelor matematice repetitive.
* **Consecvență**: Sincronizarea instantanee a datelor financiare la fiecare inserare sau actualizare.
