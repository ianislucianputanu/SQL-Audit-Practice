# Audit Tranzacții - Python & SQL

### Descrierea Proiectului
Acest proiect se concentrează pe detectarea anomaliilor în fluxurile de date externe. Folosește Python pentru a interoga baza de date și a identifica înregistrările care nu respectă regulile de autorizare.

### Scenarii de Audit Verificate
* **Erori de autorizare:** Detectarea valorilor 'NULL' sau 'ERROR' în tranzacții.
* **Integritatea datelor:** Verificarea automată a fluxurilor financiare suspecte prin scripturi de audit.

### Tehnologii
* **SQL (MySQL):** Interogări complexe de filtrare a erorilor.
* **Python:** Scripting pentru automatizarea verificărilor periodice.

### Rezultate
* **Eficiență:** Reducerea timpului de verificare a tranzacțiilor prin procesare automată în Python.
